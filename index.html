Welcome

The FCS International Project is a multinational collaboration dedicated to expanding scientific understanding of Familial Chylomicronemia Syndrome (FCS) through shared data, international participation, and evidence‑based research.

Project Overview

This project connects data and researchers across countries to strengthen knowledge of FCS, improve evidence for clinical understanding, and support future global collaboration.
